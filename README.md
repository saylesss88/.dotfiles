# To use these dotfiles, clone the repo to ~/.dotfiles. Install Gnu Stow

```bash
cd ~/.dotfiles
```

Stow the programs you want:

```bash
stow nvim
stow waybar
```

- This will place the files in the correct directories and use symlinks to this
  directory.
